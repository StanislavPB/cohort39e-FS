Написать приложение для интернет-магазин (back-end)

1. Обеспечить получение и обработку запросов пользователя со клиентской части
для оформления заказа на покупку товара

2. Обеспечить:
- хранение данных о товаре (список товаров и их количество и цены)
- хранение информации о заказчиках (идентификация и история покупок)
- хранение информации о заказах (состав заказа, его характеристики)
- хранение информации о клиентской корзине (неоформленный заказ, т.е. выбранные товары в корзине, но еще не купленные)
- регистрация нового пользователя
- авторизация пользователя для оформления заказа
- хранение данных о видах оплаты
- хранение данных о параметрах заказа (выполнен, оплачен / не оплачен, отгружен)


==============================================================================

3. Какие запросы приложение должно уметь обрабатывать:

Работа с товарами:
- передать все товары
- передать все категории
- передать товары по категории
- передать товары по категории в ценовом диапазоне
- передать список всех производителей
- передать товары по категории и производителю
- передать данные о количестве конкретного товара
- показать конкретный товар по id

Работа с пользователем:
- регистрация
- авторизация
- редактирование данных

Работа с заказами:
- показать информацию о корзине пользователя
- внести изменения в корзину
- оформить заказа (т.е. сформировать заказ и перенести туда товары из корзины,
очистить корзину, внести изменения в количество товаров на складе в зарезервированные)
- отмена заказа (по запросу пользователя)
- передать список всех заказов пользователя
- передать информацию о конкретном заказе
.......
